## Recipidea



日頃献立に悩んでいませんか。

簡単に食材を入れると、ChatGPTでAIがレシピを出してくれます。



美味しいのか、美味しくないのか。



その評価をするのはAIではありません。実際に、料理を食べる人間です。

レシピを見て料理を作り、レビューできます。



そのレビューを他の人が見てリアクションを送ります。

ChatGPTがはじき出した答えが人気の料理になるかもしれません。



ユーザー情報を求めてないのですぐ投稿やリアクションできます。
